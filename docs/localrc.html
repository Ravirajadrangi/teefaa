localrc

localrc is a user maintained settings file that is sourced at the beginning of bootstrap.sh. This gives the user's local valuables.

#!/usr/bin/env bash

Which image do you want to install? This name has to be listed on IMAGE_LIST.

IMAGE_NAME=ubuntu-12.04

Define where you mount root partition.

MNT_ROOT=/mnt

Define logfile.

LOGFILE=/tmp/bootstrap.log

IMAGE_LIST=$TOP_DIR/image.list

Define the partitioning type.

#mbr: Master Boot Recorder
#gpt: GUID Partition Table
#none: Don't partition use existing partitions
export PARTITION_TYPE="mbr"

Define the disk to install the sytem.

disk=sda

Define Size[GB], Type and Mount Point, pertition_id=(size[GB] type mount). If size is set as "-1", it means the size is the rest of the disk.

sda1=(2 swap none)
sda2=(50 ext4 "/")
sda3=(-1 ext4 "/local")

If you want to setup static IP Addresses and hostnames in the installation process, these ADDR_LIST, DIFF_PREFIX and INTERFACES Need to be difined.

DIFF_TYPE=general
DIFF_DIR=$TOP_DIR/diffs/$DIFF_TYPE/$os_DISTRO/$os_RELEASE

If DIFF_TYPE isn't general, these will be needed.

INTERFACES="eth0 eth1 ib0"
GATEWAY="149.165.146.254"
ADDR_LIST=$TOP_DIR/address.list
DIFF_PREFIX="DIFF"
