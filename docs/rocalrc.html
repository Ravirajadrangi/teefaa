localrc

source localrc

localrc is a user maintained settings file that is sourced at the beginning of teefaa.sh. This gives the user's local valuables.

#!/usr/bin/env bash

Define where you mount root partition.

DEST=/mnt

Find the hosts file. HOSTS=$TOP_DIR/hosts

HOSTS="/etc/hosts"

Define logfile.

LOGFILE=/tmp/uruma.log

REPO_LIST=$TOP_DIR/repo.list

Define interfaces. This will assume you have, in your HOSTS above, ${hostname} as EXTINTERFACE's ip address, ${hostame}i as INTINTERFACE's address and ${hostname}ib as INFINI_INTERFACE. Something like this;

HOSTS=$TOP_DIR/hosts

10.129.1.1 node-001 172.20.1.1 node-001i 192.168.1.1 node-00ib

export INT_INTERFACE="eth0"
export EXT_INTERFACE="eth1"
export INFINI_INTERFACE="ib0"

Define the disk name

export DISK=sda

Define the partitioning type. mbr: Master Boot Recorder gpt: GUID Partition Table dont: Don't partition use existing partitions testmbr: output the fdisk inputs to testpartitioning.txt testgpt: output the parted inputs to testpartitioning.txt

export PARTITION_TYPE="mbr"

Define your partitions. This example creates 2GB for swap as sda1, 50 GB with ext4 for "/"(root partition) as sda2, the rest with xfs for /var/lib/nova as sda3.

export PARTITIONINGS="2 100 -1"
export FILESYSTEMS="swap ext4 xfs"
export MOUNTPOINTS="swap / /var/lib/nova"
