Installation¶

Requirements¶

  • Python, version 2.7 ~ 2.7.6
  • Python Modules
    • Fabric >= version 1.6
    • Cuisine >= version 0.6
    • PyYAML >= version 3.10

Installation¶

  • Download Teefaa from Github:

    git clone https://github.com/cloudmesh/teefaa.git

  • Install:

    cd teefaa
python setup.py install

  • Confirm:

    teefaa --help
